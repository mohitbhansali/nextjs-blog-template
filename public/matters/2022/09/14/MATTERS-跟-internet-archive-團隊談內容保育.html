跟 Internet Archive 團隊談內容保育

Sep 14, 2022 • LikeCoin DAO

最新的 Writing NFT 文章是 ckxpress 有關 DWeb Camp 之行的體驗分享。

節錄：

經常有朋友問到如何翻查《蘋果日報》、《立場新聞》等被消失的新聞，除了功德無量的《聞庫》，Wayback Machine 是另一個方法，甚至更加原汁原味。方法也非常簡單，到 web.archive.org ，在 Collection Search 的部分下拉菜單，選擇所需的 collection 如 hk.appledaily.com（近 50 萬網頁）、thestandnews.com（近 25 萬網頁）等，輸入關鍵字，再按 Search 即可。
別以為以上的功能理所當然，事實上，Wayback Machine 現時只有 10 個合集可供用戶針對式搜索，當中香港的媒體除了以上兩個，還有 scmp.com，10 個合集整理佔了 3 個，可想而知香港多麼幸運。當然，這份備受關顧的福氣，也同時反映了香港的新聞業有多糟糕。

若你覺得這篇文章有價值，請收集它到你的 Web3 書架。

在人生的最後階段，我會以甚麼方式做記錄？

本周五（16/9）晚，我們邀請了社群的活躍成員米高 (@DrThankYouC) 及太太 Tiffany 來 LikeCoin Discord，分享利用 NFT 幫助弱勢社群的想像。他們將把首篇項目介紹的文章出版成 Writing NFT 供大家收集。

以下是他們的項目簡介：

Tiffany，一位香港自閉症人士社會企業的創辦人，她是人道年獎、HKICT Award 年度大獎的得主，從事家庭輔導及自閉症人士職涯規劃培訓超過十年，一直為將要投身社會工作的「特殊孩子」義務提供資訊科技及職場人際關係互動技巧訓練，讓這一班「來自星星」的特殊孩子 ，為自己的夢想，闖出另一片天。

然而不幸地，Tiffany 去年突然暈倒，入院後發現為腦幹大範圍缺血性中風，有70%腦幹細胞完全死亡，導致她半身不遂。經過一年的復健期，Tiffany 開始了重投社會，但體力經已大不如前，她心知未必可以再如昔日大量耗損精神地投入工作。

數年前，Tiffany 開始接觸區塊鏈的內容創作平台，在 Matters x LikeCoin 上寫社企經歷的故事，也設立了一個 LikeCoin 驗證人 FamilyOnline.TV。到了人生的最後階段，這一次可能是 Tiffany 人生最後希望可以親自實踐的一個項目，她竭力學習如何建立一個 Web 3 社區 KiiTOS Galaxy 及 創作基金 iCare Foundation，讓有特殊學習障礙的孩子也同樣可以自由平等地在網絡 3.0 的世界裡，天馬行空地探索無邊無際的創意天際。

KiiTOS Galaxy 會以 NFT Community模式運作，當中的 80% 金額，將會撥入創作基金及社企工作室的日常運作，創作基金會直接買入「弱勢社群的創作作品」，並且重新 NFT 化；亦會在 Web 3 社區持續舉辦線上的創作活動。KiiTOS Galaxy 社群的參與者稱為KiiTOS Hero；KiiTOS 是芬蘭語，具有「對人感謝感恩」的意思。我們相信，每一個人都須要有一顆「快樂」及「感恩」的細胞「KiiTOS Cell」；任何「KiiTOS Cell NFT」持有者，將會獲得 KiiTOS Moonster 星球尋寶工具 或 KiiTOS Hero 英雄角色。

Tiffany 想邀請您，一起構建 KiiTOS Galaxy 這個創意無限、大愛無疆的 Web 3.0 社區。
