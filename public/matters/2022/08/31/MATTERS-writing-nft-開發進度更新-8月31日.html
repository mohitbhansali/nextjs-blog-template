Writing NFT 開發進度更新（8月31日）

Aug 31, 2022 • LikeCoin DAO

Writing NFT beta 於 8 月 1 日推出後，功能快速迭代。

以下是本月更新的重點概覽。

改善收集流程最優先

用戶現可於 Feature 頁、liker.land 的個人主頁及個別 Writing NFT 詳細資訊頁收集 NFT。目前大部份 NFT 購買的流量均來自 Writing NFT 詳細資訊頁，使用情境是用戶於任何渠道，如社交媒體、即時通訊應用或電郵等分享 Writing NFT 詳細資訊頁的連結。例：點擊此鏈結以直接訪問該 Writing NFT 的詳細資訊頁。改善此頁用戶介面的任務因而顯得優先。

本月有關 NFT 詳細資訊頁的更新包括：

  • 顯示銷售情況資訊、改良 og 縮圖及資訊顯示等
  • 修正於手機顯示 Writing NFT 時的 UI 異常
  • 使用更易明白的錯誤訊息
  • 基本分享按鍵

一開始的 liker.land 頁面沒有連接錢包功能，後來加上了。用戶連結錢包後可查看錢包結餘、到 My Dashboard 頁面查看自己的收藏和著作、查看讚賞公民會籍狀態，及修改用戶設定等。

Writing NFT 轉贈功能

新增了轉贈 Writing NFT 的功能。此功能讓 Writing NFT 的使用場境變得更豐富，擁有 NFT 除了對自己有意義外，還可以當作心意轉贈給朋友。某些文字可能對朋友有特別深刻的意義，從前總不成把字列印出來送贈，現在 Writing NFT 提供了體面的方式了。

二級市場交易分潤機制

目前二級市易交易的功能尚未完整，但初步的功能已於第一筆交易時實現，日後支援 NFT 轉手功能時，將按此機制分潤。

目前的分潤方式是：

  • 平台手續費 2.5% （由 Liker Land 收取）
  • 持份者 10% （由內容的 ISCN 擁有者定義誰為持份者）
  • 賣家 87.5%

本月最具代表性的案例為章濤的《財科暗戰序》 Writing NFT，每筆交易收益的 10% 將分給 7 位持份者。

此機制尚在實驗中，日後很大機會作出修改。

兼容更多網站內容

本月嘗試包括把 WordPress （ckxpress.com）、Substack（區塊勢）及任意平台（章濤及薯伯伯的個人網站）出版成 Writing NFT，曾嘗試把 Patreon 文章出版但未能成功。

正在實驗中的還有 Matters 和 Medium 上的文章。Matters 文章由於已支援 ISCN，已證實能順利出版成 Writing NFT；Medium 文章的實驗則預計於下個月內見結果。


LikeCoin Writing NFT 開放整合任何寫作平台，希望未來能支援更多不同的站點。

其他有關 Writing NFT 的小工具

雖然 Writing NFT 的交易資訊都是公開的鏈上數據，但沒有好的用戶介面工具很難查看。感謝 wancat 幫忙開發了一個 NFT Dashboard 小工具，以查看例如排行榜、社交關係、交易額等數據。此工具是開源項目，歡迎任何有心人共同開發改善。

另外，wancat 也開發了一個 ISCN browser，這工具除了可以瀏覽 ISCN 記錄以外，還可用來修改 ISCN 記錄，常用的功能例如修改持份者資料或轉移 ISCN 擁有權。

作者白名單安排

系統目前仍有不少需要修正的問題，暫未能全面開放，請大家耐心等候。目前我們收到超過 100 名作者登記白名單，會逐一邀請試用 Writing NFT 的功能，收集反饋意見，改善產品。
