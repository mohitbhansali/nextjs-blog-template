hex- 與「六個之；六個數；六角形；六角之；六個；六數；六之；六；house樣介形；陸」等的橋接

Nov 8, 2022 • 羅聖爾

hex- = heks (實際發音) = HE個之 = 朋ε個之 = 朋3個之 = 朋三個之 = 三三個之 = 六個之。其中 H 約略象徵漢文「朋」，而「朋三」(即類似「雙三」之意) 所轉換的「三三」應是指「三加三」的意思；此時 k 約略形聲華語「個」的起音。
hex- = heks = HE個數 = 朋ε個數 = 朋3個數 = 朋三個數 = 三三個數 = 六個數
hex- = heks = HE角形 = 朋ε角形 = 朋3角形 = 朋三角形 = 三三角形 = 六角形。此時 k 約略形聲台閩語「角」的起音。
hex- = heks = HE角之 = 朋ε角之 = 朋3角之 = 朋三角之 = 三三角之 = 六角之
hex- = heks = 合E個雙 = 合ε個雙 = 合3個雙 = 合3個雙 = 合三個雙 = 合三個三個 = 六個
hex- = heks = HE結數 = 朋ε結數 = 朋3結數 = 朋三結數 = 三三結數 = 六數。此時 k 約略形聲台閩語「結」的起音而作爲構字用語。
hex- = heks = 朋Ε加之 = 朋ε加之 = 朋3加之 = 朋三加之 = 三三加之 = 六之。此時 k 約略形聲台閩語「加」的起音而作爲構字用語。
hex- = heks = 合Ε加三 = 合ε加三 = 合3加三 = 合三加三 = 六
hex- = heks = house樣介形 = 房舍樣介形 = 六。此時其中的「房舍樣介形」即約略描述漢字「六」的大多數甲骨文字體形狀，其中 k 約略形聲台閩語「介」的起音，而漢字「介」在此處可能被同時銓釋爲一棟房舍的象徵而代表 house，再以 house 的字首 h 作爲 hex- 的字首。
hex- = heks = 人一結雙 = 人丨結雙 = 人丨丨= 六，此時 h 約略象徵漢文「人」的甲骨文字體，而其中的「人丨丨」也是約略描述漢字「六」的大多數甲骨文字體形狀。此外，爲何漢語「六」的起音也很接近漢語「人」的起音 (尤其是台閩語)，在此似乎可以找到一個線索，至於是否也和 Holy Bible 所記載的「God 在創世記的第六天造出人類」的傳說有點神秘的關聯，筆者就不太清楚了。
hex- = heks = 阜earthπ十 = 阝土兀十 = 阝土圥 = 陸，亦即 hex 也能橋接於漢字「六」的大寫形式「陸」。其中 h 約略形聲漢語「阜」的 fù 式發音之起音但轉爲 h 音，k 轉 90 度約略象徵漢文「兀」。
前述的字根 hex- 源於 English 式拼音之 Greek 的 hex 一字，並且在化學的術語中也經常會看到這類字根。(P.S. 參考〈six 與「六；陸；六數；六計數；六之；六的；六个；六個；六點；六時；六結一」等的橋接，兼談 June〉-- https://vocus.cc/article/62cb8349fd8978000171aea7)
P.S.
hexa-：六個之樣；六個數的；六角形樣；六角形的；六個的；六數的；六之樣；六樣；六的；六之；六个，意通「六個」；六；六角；六言；六行；六土；六地；六邑，意通「六國」；六爻，此時爲漢字古籍《易經》的用語，即 hexa = 人E乂2 = 𠆢L二乂乂 = 𠆢連II爻 = 𠆢連丿丨爻 = 介爻 = 六爻，其中漢文「介」約略象徵漢字「六」的早期象形文 (P.S. 參考〈six 與「六；陸；六數；六計數；六之；六的；六个；六個；六點；六時；六結一」等的橋接，兼談 June〉-- https://vocus.cc/article/62cb8349fd8978000171aea7)，相當於一個「卦」。
hexagon：六個之樣記號；六個數的記號；六個數的角樣；六角形樣記號；六角形的記號；六個的記號；六數的記號；六角樣；六角式；六角形。類似「六邊形」或「六角形」之類。
hexagonal：hexagon的；hexagon樣容；hexagon一類。或縮寫爲 hex.。
hexagram：六個之樣刻畫樣；六個數的刻畫樣；六個的刻畫樣模；六個數的角銳形；六角形的角銳形；六個的角銳樣；六數的角銳樣形；六爻卦象、六爻卦圖、六爻卦符、六爻合結卦(象)，亦即包括漢字古籍《易經》中帶有「六爻」之圖形組合結構的「卦」、「卦符」、「卦象」、「卦圖」，也常用此字來表達。
hexadecimal：六合十進位法，意通「十六進位法」，類似中文漢字數學中的「十六進位制」或「十六進制」，其中 a 代表 and 或漢字「合」，d 或 de 或 dec 與 decade (十) 相關；六和十的，意通「十六的」；六和十第，意通「第十六」；六和十角(的)，意通「十六角形(的)」。
hex：此字若視爲 English 式拼音之 Greek 單字 hex，就與前述 hex- 的意思相同，但一般是作爲 hexagon 或 hexadecimal 的簡縮寫。不過，在 English 中，hex 一字還可能有其它撞字的另義，例如：法技施，類似「施法術」；惑，即 hex = heks = 或結心 = 惑，此義可作動詞，但作名詞使用時也可能包含各種「惑人之事物」的指稱；惑困，意通「困惑」；混搞，意通「搞混」；糊搞，類似「弄迷糊」；昏搞，類似「搞昏頭」；難以搞之、難以行之、難以進之，此時字首 h 與 hard 相關；發咒；哼咒。
hexad：六值、六計，類似「六數」；六個；六塊；六个一堆、六个一隊、六个一團、六个一體，類似「六個一組」。此字爲名詞。
hexaemeron：六日造世論，類似「六日創世論」，即 hexaemeron = hexa.e.m.e.ron = 六.E.make.E.論 = 六.日.造.l三.論 = 六.日.造.兩III.論 = 六.日.造.二III.論 = 六.日.造.丗.論 = 六.日.造.世.論 = 六日造世論。其中漢字「丗」旁通漢字「世」，所以由「丗」轉注「世」。
hexahedral：六面體的。其中 -hedral = 環eye歹豊al = 〇目骨豊的 = 囗目體的 = 囬體的 = 面體的，而漢文「歹」演化自「歺」，本是來自一塊骨板的象形，所以可能由漢文「歹」通過漢文「歺」以轉注漢文「骨」。
hexahedron：六面體(單元)。
hexangular：六角的；六角樣；六角樣的；六角形樣；六角形樣的；六角樣形的。
hexapartite：六个部一團之；六个分開之；六分之。
hexapla：六言編聯；六言編列；六言並列；六言版本；六言版。意指「六種語言編列在一起的版本」。
hexapody：六言詩；六言絕句。
hexastich：六行詩
hexapod：六个足的；六个足虫，此時多指「有六足的昆蟲」。
hexarchy：六邑結義，類似「六國聯盟」；六地結義；六地聯結。
