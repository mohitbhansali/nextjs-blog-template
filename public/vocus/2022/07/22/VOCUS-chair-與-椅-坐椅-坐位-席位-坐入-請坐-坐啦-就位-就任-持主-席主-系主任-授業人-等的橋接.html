chair 與「椅；坐椅；坐位；席位；坐入；請坐；坐啦；就位；就任；持主；席主；系主任；授業人」等的橋接

Jul 22, 2022 • 羅聖爾

chair = ch.ai.r = 奇.A圓.木 = 奇.亼〇.木 = 奇. 亼口.木 = 奇.合.木 = 奇合木 = 椅，其中 ch 約略形聲華語「奇」的起音，「合」僅作構字用語，r 同 root (根) 之 r 轉換漢文「木」。
chair = 雙人1椅二 = 人人丨椅二 = 坐椅，其中 c 通過字母本音約略形聲漢語「雙」的起音而作爲構字用語，h 約略象徵漢文「人」的早期象形文，i 約略形聲漢語「椅」再轉音。
chair = ch.a.i.r = 坐.A.ㄧ.亻= 坐.Λㄧ.ㄧ.亻= 坐.人ㄧ.ㄧ.亻= 坐.大.ㄧ.亻= 坐.大ㄧ亻= 坐.位 = 坐位。此時 ch 約略形聲台閩語「坐」的起音再稍微轉音，而漢字「人」在古代也有類似 Greek 字母 Λ 的字形，並且其中漢文「大一」的上下組合即約略描述漢文「立」的甲骨文字體。
chair = c.h.a.i.r = 席.合.A.ㄧ.亻= 席.合.Λㄧ.ㄧ.亻= 席.合.人ㄧ.ㄧ.亻= 席.合.大.ㄧ.亻= 席.合.大ㄧ亻= 席.合.位 = 席位。此時有點類似 seat 之意，但主要應是指椅子的席位。其中 c 通過字母本音約略形聲漢語「席」的起音，漢字「人」在古代也有類似 Greek 字母 Λ 的字形，而其中漢文「大一」的上下組合即約略描述漢文「立」的甲骨文字體。(P.S. 參考〈seat 與「席地」、「席點」、「席墊」、「席毯」、「蓆氈」、「席單」、「坐蓆」、「席位」等的轉換密碼，兼談 sit 與「席坐」或「坐」的橋接關係〉-- https://vocus.cc/BridgeWords/5cde1590fd89780001a19329)
chair = 雙人1冫入 = 人人丨二入 = 坐入，此時即「入坐」的同義倒裝詞。
chair = 請A冫人 = 請Λ一二人 = 請土人人 = 請坐，或意通 sit down please。其中符號「Λ一」的上下組合約略象徵漢文「土」的甲骨文字體之一。
chair = 坐啦，此時有點直接約略形聲台閩語「坐啦」的感覺，或意通「請坐」。
chair = 就A一亻= 就Λ一一亻= 就人一一亻= 就大一亻= 就立亻= 就位
chair = 京and尤任 = 京和尤任 = 就任，可能是「就位」的引申義。
chair = ch.air = 持.一丶丨二 = 持.主 = 持主，此時爲「主持」的同義倒裝詞。其中 ch 約略形聲漢語「持」的起音。這類意義又常作爲 chairman、chairwoman、chairperson、chairmanship 等與「席主」或「主席」之類意思相關的單字之構字元素。(P.S. 參考〈chairman 與「洽合議理民人」或「洽會主管人」或「持主人」意即「會議主持人」、「主持人」、「議長」或「主席」的轉換密碼〉-- https://vocus.cc/BridgeWords/5cdc8e89fd89780001453bfc)
chair = c.h.air = 席.合.主 = 席合主 = 席主，此時爲「主席」的同義倒裝詞。其中 air 同上式轉換漢字「主」，「合」僅作構字用語。耐人尋味的是，上式和此式連續兩個橋接式都發現 air 這個拼音也能橋接於漢字「主」，那可見常常被提到的生命三要素「陽光、空氣、水」中的「空氣」還真的是生命的三個主要的要素之一。莫非所謂的「主持」或「主席」也有一種特殊的能力，可以把所在之處的空間氣氛搞得很不錯？
chair = cHair = 系朋air = 系airair = 系air.air = 系.主.合亻壬 = 系.主.任 = 系主任，據說多指大學的系主任。其中 c 通過字母本音約略形聲華語「系」的起音，h 通過大寫 H 約略象徵意會漢字「朋」而作爲意通漢字「雙」的構字用語。
chair = 授業人，可能類似「教授」或古人所謂的「師者，所以傳道、授業、解惑也」(P.S. 語出唐代韓愈的《師說》) 的內涵之一，據說多指大學教授的職位。可見，大學教授的壓力可能也不小，因爲此時或許已經到了要給學生「授業」的階段，也許是一種事關學生將來是否得以順利就業發展的壓力，也因此可能有些大學就會安排一些實習課程或建教合作的方案，讓學生感受一下未來就業發展的可能氣氛。這麼說的話，搞不好「解惑人」是更高的階段也說不定，例如碩士班或博士班的師者。不過如果不是那麼講究精確發音的話，也許 chair 也能橋接於意通「傳道人」的「傳路人」，以及「解惑人」，從而構成較完整的韓愈版「師者」之概念。

P.S.

chairlift：坐椅升降台、坐椅升降通，類似「坐椅式昇降機」或「坐椅式電梯」；坐椅纜車，例如台灣的貓空纜車也是一種 chairlift，因爲纜車的車廂內部設有坐椅，但滑雪勝地則比較常見坐椅外露而沒有車廂外殼包覆的 chairlift，此時其中 lift = 纜withF田 = 纜與丨二田 = 纜與一II田 = 纜與卄田 = 纜與十十田 = 纜與車 = 纜車，亦即「纜車」也是一種帶有「升降通」性質的交通工具，而一般大樓裡的「昇降機」或「電梯」也可能說是一種垂直昇降的纜車，因爲有很多電梯是利用鋼纜或鋼索來拉升垂降搭載貨物或乘客的機箱。
