stop = 豎亭和point =丨亭和㇀= 停。漢字「停」在古代字書的記載裡，有「行中止」(可能包括「行動中停止」、「行途中停止」、「行至目標位址」) 和「定」(定於所在、固定於所在之處) 的意思。巧合的是 stop 也能橋接於漢字「行中止」與「定」，即：stop = 行直口ph = 行丨口f = 行中foot = 行中止；stop = 雙點偶ph = 二丶二f = II丶一一foot = 宀一止 = 宀一龰 = 定。
stop = 豎直one平 =丨丨一一 = 丨Iㄧ一 = 止。此「止」字類似「停」之意，但也可能是「址」的本字，東漢許慎《說文解字》便記載「止，下基也。象艸木出有址，故以止爲足。」，也就是漢字「止」有時意通漢字「址」，有時又意通漢字「足」，所以也可能意通 English 的 address 或 foot。
stop = 豎兜偶ph =丨兜二f = 土foot = 土止 = 址，此時意通 address (址；地址；位址；網址；信址；名址；址寫；址署；名署；稱呼；……) 的含意之一。亦即，我們常聽說的 bus stop 或「公車站」也可能說是「公車址」。

stop 這個單字，看起來好像和 step 一字是兄弟或姐妹一般，兩字只差在 e 和 o 的不同。step 的 e 若以大寫 E 來看，好像畫出了一個連續過程中的好幾個階段；stop 的 o 則好像帶有句點的中止意像。頗爲耐人尋味！…… (P.S. 參考〈step 與「步；行步；走步；跕步；躡步；踏；踏步；踩；踩步；踩踏；跨；跨步；跨足；跨踩；伐步；跳步；行跳步；行蹈步；行蹈舞；行動移步；行動一步；行動之步；行台步；行踏步；行踏印；踏印；足印；足跡；跡；踏板；階；踏階；台階；階段；級；級踏；階級；級階；級等；級調；驟步；逐步；調步；聲調步；聲調部；聲階；測步；繼；後；後繼；異」等的橋接〉-- https://vocus.cc/article/62df37fdfd897800016c2fed)
漢字古籍《禮記.大學篇》的第一段文字寫道「大學之道，在明明德，在親民，在止於至善。知止而后有定，定而后能靜，靜而后能安，安而后能慮，慮而后能得。物有本末，事有終始，知所先後，則近道矣。」，似乎也講到了認知 step 和 stop 兩者的重要性，不過這段文字是把「大學之道」的「止」定義為「至善」(stop = st偶平 = stst平 = ㄙ土善兜一 = 至善)，頗爲耐人尋味！…… 難怪，best 一字好像也能同時橋接於漢字「大學之道」與「至善」二詞，即 best = 平Es道 = 一L二s道 = 一人ss道 = 大學之道；best = lo一善土 = Ldot一善土 = ㄥ點一善土 = ㄥ丶一善土 = 厶一善土 = 至善。
